# Latent Degradation Representation Constraint for Single Image Deraining (ICASSP 2024)
# Requirements
* Linux Platform
* NVIDIA GPU + CUDA CuDNN
* PyTorch == 1.13.1
* Python 3.7.0
* torchvision 0.14.1
* matplotlib
* scikit-image
* opencv-python
* yacs
* joblib
* natsort
* h5py
* tqdm
